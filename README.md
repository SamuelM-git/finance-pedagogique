# 🏦 Finance LudiK

Application pédagogique développée avec Streamlit pour apprendre à gérer son argent de manière simple, interactive et ludique.

## 🎯 Objectif

Permettre aux utilisateurs de :

* Comprendre leur budget
* Développer une épargne
* Découvrir les bases de l’investissement
* Tester leurs connaissances avec un quiz

⚠️ Cette application a une finalité exclusivement pédagogique.
Elle ne constitue pas un conseil financier.

---

## 🧩 Fonctionnalités

### 📊 Module Budget

* Analyse des revenus et dépenses
* Calcul du reste mensuel
* Visualisations graphiques
* Génération d’un rapport PDF

### 💰 Module Épargne

* Simulation d’épargne mensuelle
* Projection avec intérêts composés
* Objectifs d’épargne
* Rapport PDF téléchargeable

### 📈 Module Investissement

* Simulation d’investissement
* Profil investisseur
* Impact de l’inflation
* Comparaison de placements

### 🧠 Quiz

* Questions adaptées au niveau
* Système de score
* Gamification (XP + badges)
